# SMANSASOO CBT LOCK 2.0
## SYSTEM ARCHITECTURE DOCUMENTATION

---

## 1. OVERVIEW

SMANSASOO CBT LOCK 2.0 adalah sistem **realtime CBT monitoring & security layer** yang dirancang untuk:

- Mengawasi perilaku siswa saat ujian CBT (Moodle)
- Mendeteksi pelanggaran secara realtime
- Mengelola OTP unlock system (global & master)
- Menyediakan dashboard Panitia & Pengawas
- Sinkronisasi data melalui Firebase + Vercel API
